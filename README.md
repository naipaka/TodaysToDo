# Today's ToDo iOS-APP
![Icon](https://github.com/naipaka/TodaysToDo/tree/images/icon.png)
  
## What's this？
「Today's ToDo」というiOSアプリの開発リポジトリです。
  
## Overview
タスクを管理することができるアプリです。  
通知機能、1日に登録できるタスクの数の制限等を用いて確実な実行を促しています。
